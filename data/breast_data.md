# Breast MRI Dataset Source
This project was trained and evaluated using the following dataset:

Dataset: Breast MRI Tumor Classification Dataset
Source: https://www.kaggle.com/datasets/abenjelloun/breast-mri-tumor-classification-dataset

About the data: The images used were collected from The Cancer Imaging Archive (TCIA):https://www.cancerimagingarchive.net/collection/breast-diagnosis/
Only MRI scans were picked out from this vast collection, preprocessed, and filtered. The following 2 classes are included: Benign and Malignant.

To Download:
Create a Kaggle account
Accept the dataset terms
Download as ZIP
Place into: /private-repo/data/breast_mri/
