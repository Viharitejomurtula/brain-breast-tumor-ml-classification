# Brain MRI Dataset Source

This project was trained and evaluated using the following dataset:

Dataset: *Brain Tumor Classification (MRI)*  
Source: https://www.kaggle.com/datasets/sartajbhuvaji/brain-tumor-classification-mri/data

To download:
1. Create a Kaggle account
2. Accept the dataset terms
3. Download as ZIP
4. Place into: `/private-repo/data/brain_mri/`
